Allows capture of requests for a toy drive. View entries. Export spreadsheet.

== Description ==

Toy Drive Community Edition provides the ability to capture, through an expanding form, the information for requesting families.

You are able to set the total number of toys requested in the drive after which the window is closed. You may also close the window manually at any time.

Includes custom form title, thank you message and window closed notice.

View submissions in spreadsheet form or download the results spreadsheet.

== Frequently Asked Questions ==

= How many children in a family can be entered? =

The form is expandable, so there is no limit to the number of children that can be added per adult
